# Projeto 07 – Criando um Relatório de Vendas e Lucros com Data Analytics no Power BI

## Descrição
Projeto de atualização de um relatório de vendas e lucros com foco na **experiência do usuário**, aplicando conceitos de posicionamento, contraste, proporção áurea e segmentação dos dados.  
Todo o trabalho foi realizado diretamente no Power BI Desktop, aprimorando visuais, organização e navegação.

## Etapas do Projeto

1. **Entendendo o desafio**
- Revisão geral do relatório original.

2. **Adequando Relatório – Parte 1**
- Ajustes iniciais do layout e visuais.

3. **Adequando Relatório – Parte 2**
- Continuação das adequações e reorganização dos elementos.

4. **Realizando considerações sobre os visuais e modificando gráfico de área**
- Revisão dos gráficos e modificação do gráfico de área.

5. **Criando matriz de descrição de vendas**
- Construção da matriz para detalhamento das vendas.

6. **Ajustando estilização dos visuais e página**
- Melhoria da estética, alinhamentos e paleta de cores.

7. **Botões de navegação e próximos passos**
- Implementação dos botões de navegação e definição dos próximos ajustes.

## Observações
- O foco do projeto é melhorar a experiência do usuário no relatório.  
- Os princípios utilizados não são regras fixas; entender quando aplicá-los e quando flexibilizá-los torna o relatório mais criativo e eficiente.

## Arquivos
- `relatorio_vendas_modificado.pbix` – Arquivo do Power BI Desktop com o relatório revisado.  
- `relatorio_vendas_modificado.pptx` – Arquivo em Power Point, exportado a partir do Workspace do Power BI Web.

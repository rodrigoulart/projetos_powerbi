# Projeto 08 – Criando Relatórios Dinâmicos com o Uso de Parâmetros no Power BI

## Descrição
Projeto prático focado na criação de relatórios dinâmicos no Power BI utilizando **parâmetros de campos**, explorando personalização, interatividade e storytelling.  
Este projeto utiliza o mesmo relatório desenvolvido no **Projeto 07**, acrescentando uma nova página contendo visuais criados com parâmetros baseados em **categorias e valores**, mantendo toda a identidade visual já construída anteriormente.

## Etapas do Projeto

1. **Entendendo o desafio**  
   Revisão do relatório anterior e definição da visão dinâmica que seria adicionada.

2. **Criando parâmetros baseados em categorias**  
   Desenvolvimento do primeiro visual dinâmico utilizando um parâmetro que alterna entre diferentes categorias.

3. **Criando parâmetros baseados em valores**  
   Criação do segundo visual dinâmico alternando entre métricas como *Profit*, *Sales* ou outras medidas relevantes.

4. **Aplicando estilização e identidade visual**  
   Manutenção da mesma estética do relatório original, seguindo padrões de layout, tipografia e cores.

5. **Desenvolvendo a história dos dados (Storytelling)**  
   Construção de uma narrativa clara e objetiva para apresentar os insights ao cliente.

6. **Criando a página final com parâmetros**  
   Inclusão de uma nova página no relatório, contendo visuais dinâmicos controlados por parâmetros de categorias e valores.

## Observações
- O relatório é uma evolução direta do **Projeto 07**, mantendo toda a base visual e estrutural.  
- A nova página deve contar uma história clara e voltada para a tomada de decisão.  
- Os parâmetros foram aplicados seguindo as técnicas apresentadas no curso.  
- A estilização deve manter consistência com os projetos anteriores.

## Arquivos
- **relatorio_com_parametros.pbix** – Arquivo do Power BI Desktop contendo o relatório com a nova página de parâmetros.  
- **relatorio_com_parametros.pptx** – Arquivo em Power Point, exportado a partir do Workspace do Power BI Web.

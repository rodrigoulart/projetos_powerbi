# Projeto 02 - Relatório Gerencial de Vendas

## Descrição
Relatório gerencial detalhado baseado na amostra “Financials” do Power BI, proposto pela Tech Education Specialist.  
O objetivo foi desenvolver um dashboard com métricas avançadas de vendas, lucro e unidades vendidas.

## Principais Implementações
- Layout organizado com navegabilidade aprimorada.  
- Segmentadores e botões de navegação personalizados, incluindo imagens e indicadores.  
- Página inicial com indicadores de desempenho e filtros interativos.  
- Página detalhando métricas de vendas, lucro e unidades vendidas.  
- Criação de medidas DAX para análise gerencial.  
- Publicação no Power BI Service e exportação para PowerPoint.

## Arquivos
- `relatorio_gerencial.pbix` - Relatório interativo no Power BI  
- `relatorio_gerencial.pptx` - Exportação do relatório para PowerPoint

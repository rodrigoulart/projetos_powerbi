# Projeto 01 - Análise de Dashboard de Vendas

## Descrição
Projeto baseado em dataset disponibilizado pela Tech Education Specialist no GitHub.  
O objetivo foi construir um relatório de vendas replicando o modelo apresentado e adicionando melhorias visuais.

## Principais Implementações
- Criação de dashboards interativos com mapas de vendas e lucro por país.  
- Gráfico de pizza exibindo lucro por segmento.  
- Organização e renomeação de visuais para melhor interpretação.  
- Configuração de tooltips e filtros interativos.  
- Publicação do relatório no Power BI e exportação para PowerPoint.
